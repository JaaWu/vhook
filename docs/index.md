# ``v3hook``

---

``v3hook`` 是基于 ``vue3.0`` 的 ``Composition API`` 开发的组合函数库



## Inspiration

这个项目的灵感来源于一些优秀的开源 ``react Hooks``，如： [``react-use``](https://github.com/streamich/react-use)、[``ahooks``](https://github.com/alibaba/hooks)等。



## Install

```bash
npm install v3hook
yarn add v3hook
```
